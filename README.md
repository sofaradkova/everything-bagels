# 🥯 Animated bagel shop website
A multi-page website for a local bagel shop built with Next.js and Tailwind. The redesign is aimed to improve its online presence through carefully crafted interactions and cohesive customer journey.

## 🛎️ Demo
https://everything-bagels.vercel.app/

## 💡 The story
It makes me sad to see a cafe with the most delicious fusion of bagels and asian cuisine have a website consisting of a confusing menu and an ‘order’ button. Taking this matter into my own hands, I have fully restructure their UI, retelling the story of their menu in a new light. Every item now has an appealing visual representation instead of the plain text description, while the website itself is filled with animations, making the customer experience intuitive, interactive, and playful.

## 🔮 Features
- **Animated UI** – Smooth page transitions and micro-interactions built with Framer Motion
- **Visual Menu Experience** – Menu items displayed as rich image cards instead of text lists
- **Story-Driven Layout** – Landing page designed to highlight the unique fusion of bagels and Asian cuisine
- **Reusable Component System** – Modular React components for menu items, sections, and navigation
- **Smooth Navigation** – Intuitive page structure improving the customer browsing journey

## 🛠 Tech Stack
- Framework: Next.js
- Frontend: React
- Styling: Tailwind CSS
- Animations: Framer Motion
- Deployment: Vercel

## 🎨 Design Decisions
This redesign focuses on transforming a static restaurant website into an engaging browsing experience.

Key ideas behind the redesign:
- Replace text-heavy menus with visual food cards
- Use motion and interaction to make browsing playful
- Create a clear customer journey from landing page → menu exploration → ordering
- Maintain fast performance and simplicity despite rich animations
